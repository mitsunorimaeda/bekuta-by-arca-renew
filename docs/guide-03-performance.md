# 【選手向け】パフォーマンス測定ガイド

Bekutaでは、様々なパフォーマンス測定の結果を記録して、成長を可視化できます。
自己ベストを更新するとお祝い通知が届きます！

---

## パフォーマンス記録の入力方法

1. メニューから **「パフォーマンス」** を選択
2. **測定種目** を選ぶ
3. 測定値を入力
4. **「記録する」** をタップ

---

## 測定カテゴリ

### ジャンプ測定

跳躍力を測定します。

| 種目 | 測定値 | 中高生の参考値 |
|------|--------|---------------|
| CMJ（カウンタームーブメントジャンプ） | 跳躍高(cm) | 25〜50cm |
| CMJ（腕振りあり） | 跳躍高(cm) | 30〜55cm |
| スクワットジャンプ（SQJ） | 跳躍高(cm) | 20〜45cm |
| 立ち幅跳び | 距離(cm) | 150〜250cm |
| 立ち5段跳び | 距離(cm) | 800〜1400cm |
| DJ RSI（ドロップジャンプ） | 跳躍高・接地時間から計算 | RSI 0.8〜2.0 |
| RJ RSI（リバウンドジャンプ） | 平均跳躍高・接地時間から計算 | RSI 0.8〜2.0 |

**RSI（Reactive Strength Index）** = 跳躍高 ÷ 接地時間
バネの強さを表す指標です。高いほど効率よくジャンプできています。

### 筋力測定

主要なリフト種目の記録です。推定1RM（1回だけ挙げられる最大重量）が自動計算されます。

| 種目 | 入力値 | 計算される値 |
|------|--------|-------------|
| ベンチプレス | 重量(kg) × 回数 | 推定1RM |
| バックスクワット | 重量(kg) × 回数 | 推定1RM |
| デッドリフト | 重量(kg) × 回数 | 推定1RM |
| ブルガリアンスクワット（左/右） | 重量(kg) × 回数 | 推定1RM |
| 懸垂 | 回数 | — |

**体重比（相対1RM）** も自動計算されます。体重記録があれば、自分の体重に対してどれくらいの重量を扱えているかがわかります。

### スプリント測定

短距離のタイムを記録します。

| 種目 | 参考値（中高生男子） |
|------|---------------------|
| 5m走 | 1.0〜1.3秒 |
| 10m走 | 1.7〜2.1秒 |
| 20m走 | 3.0〜3.6秒 |
| 30m走 | 4.0〜5.0秒 |
| 50m走 | 6.5〜8.0秒 |

### 持久力測定

全身持久力の指標です。

| 種目 | 入力値 | 算出される値 |
|------|--------|-------------|
| 20mシャトルラン | 往復回数 | 推定VO2max (ml/kg/分) |
| Yo-Yo Test IR1 | 走行距離(m) | 推定VO2max |
| Yo-Yo Test IR2 | 走行距離(m) | 推定VO2max |
| クーパーテスト（12分間走） | 走行距離(m) | 推定VO2max |
| 1000m走 | タイム（分:秒） | — |
| 1500m走 | タイム（分:秒） | — |

### アジリティ測定

方向転換能力を測定します。

| 種目 | 説明 |
|------|------|
| アローヘッド（左/右） | 矢印型のコースを走るタイム |
| プロアジリティ（左/右） | 5-10-5テスト |
| 0-5-0（左/右） | 5m往復ダッシュ |
| 反復横跳び | 20秒間の反復回数 |

## 自己ベスト更新

パフォーマンス測定で自己ベストを更新すると：

- 画面にお祝いメッセージが表示されます
- 自己ベストの一覧で確認できます
- コーチにも通知が届きます

定期的に測定して、自分の成長を数値で実感しましょう！

---

## 測定のコツ

1. **同じ条件で測定する**: 時間帯、ウォーミングアップ、測定機器を揃える
2. **定期的に測定する**: 月1〜2回が目安
3. **コンディションが良い日に**: 疲労が溜まっている日は正確な値が出にくい
