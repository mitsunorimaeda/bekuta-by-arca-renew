# 【選手向け】ACWRって何？怪我リスクの見方ガイド

「ACWRが高い」「赤信号」—— コーチからこう言われたこと、ありませんか？
このガイドでは、ACWRの意味と、自分のデータの見方をわかりやすく解説します。

---

## ACWRとは？

ACWR = **Acute:Chronic Workload Ratio（急性・慢性ワークロード比）**

簡単に言うと：

**「最近の練習量」÷「普段の練習量」**

- **急性負荷（Acute）**: 直近7日間のトレーニング負荷の合計
- **慢性負荷（Chronic）**: 直近28日間のトレーニング負荷の1日平均

この比率で「普段と比べて、最近の練習がどれくらいキツいか」を数値化しています。

---

## ACWRの目安

| ACWR値 | 状態 | 意味 |
|--------|------|------|
| **0.8〜1.3** | 理想的（グリーン） | 普段通りの練習量。最もパフォーマンスが出やすく、怪我のリスクも低い |
| **0.5〜0.8** | やや低い（ブルー） | 練習量が少なめ。休養期間や怪我からの復帰期かも |
| **1.3〜1.5** | 注意（イエロー） | 急に練習量が増えている。怪我に注意が必要 |
| **1.5以上** | 高リスク（レッド） | 怪我のリスクが非常に高い。練習量を調整すべき |

---

## なぜACWRが重要なの？

スポーツ科学の研究で、**ACWRが1.5を超えると怪我のリスクが2〜5倍に跳ね上がる**ことがわかっています。

よくある危険なパターン：

### パターン1：テスト明けの追い込み
- テスト期間中は練習なし → 慢性負荷が下がる
- テスト明けにいきなり全力練習 → 急性負荷が急上昇
- → ACWRが跳ね上がる → 怪我のリスク大

### パターン2：大会前の調整ミス
- 大会に向けて急に練習を増やす
- 身体が追いついていない状態で高負荷
- → 筋肉や関節に過度な負担

### パターン3：怪我からの復帰
- 怪我で長期離脱 → 慢性負荷がほぼゼロ
- 復帰後すぐに通常メニューに参加
- → ACWRが急上昇 → 再受傷のリスク

## Bekutaでのチャートの見方

ホーム画面のACWRチャートで確認できる情報：

- **折れ線グラフ**: ACWRの日別推移
- **色付きゾーン**: 緑（理想）/ 黄（注意）/ 赤（危険）の範囲
- **急性負荷・慢性負荷**: それぞれの数値も表示

### チャートの使い方

1. **緑ゾーンをキープ**することを目標に
2. **急な上昇**（特に2〜3日で0.3以上の上昇）に注意
3. **長期間の低い値**は体力低下のサイン

---

## ACWRを理想的に保つには

1. **練習量を急に増やさない**: 週あたり10%以内の増加が目安
2. **休養日も記録する**: RPE 0で記録することで正確な計算に
3. **毎日チェックインする**: データが欠けるとACWRが正確に計算できない
4. **長期休み明けは段階的に**: いきなり全力ではなく、徐々に負荷を上げる
5. **コーチと相談する**: ACWRが黄色や赤になったら、練習メニューの調整を相談

## よくある質問

**Q: ACWRが表示されません**
A: ACWRの計算には最低7日分のチェックインデータが必要です。まずは1週間、毎日記録を続けてください。

**Q: 休養日を記録しないとどうなりますか？**
A: 休養日の記録がないと、その日の負荷が「不明」になり、ACWRが正確に計算されません。RPE 0、時間 0分で記録しましょう。

**Q: ACWRが高いときはどうすればいいですか？**
A: 練習の強度や量を少し落としましょう。完全に休む必要はありませんが、軽めのメニュー（RPE 3〜4程度）に調整することをおすすめします。コーチに相談してください。
