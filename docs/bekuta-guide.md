# Bekuta（ベクタ）総合利用ガイド — 目次

## Bekutaとは？

Bekutaは、スポーツチーム向けのコンディション管理アプリです。
トレーニング負荷（ACWR）を科学的に管理し、怪我のリスクを可視化します。

選手は毎日の記録を入力するだけ。コーチはダッシュボードでチーム全体の状態を一目で把握できます。

機材不要。スマホだけで、今日から始められます。

## 選手向けガイド

### 0. Bekutaの始め方

コーチからBekutaの招待リンクを受け取ったら、3分でセットアップ完了です。

1. 招待リンクを開く
2. 名前・メール・パスワードを入力して「参加する」
3. ホーム画面に追加（iPhone: 共有→ホーム画面に追加 / Android: インストール）
4. 通知を「許可」する
5. プロフィールを入力する

詳しくは **[Bekutaの始め方ガイド]** をご覧ください。

### 1. 毎日のチェックイン（一番大事！）

画面右下の青い **＋ボタン** をタップすると、「今日の記録」画面が開きます。

#### 練習記録
- **RPE（主観的運動強度）**: 今日の練習のきつさを0〜10で選択
  - 0 = 休養日（練習なし）
  - 5 = きつい
  - 10 = 限界
- **練習時間**: スライダーで分単位で設定
- **トレーニング負荷**: RPE × 時間 で自動計算されます

#### 体重
- 毎日同じ時間帯（朝起きてすぐがおすすめ）に測定して記録

#### 睡眠
- 睡眠時間と質（星1〜5）を記録
- 十分な睡眠はパフォーマンスの土台です

#### モチベーション
- モチベーション・エネルギー・ストレスをそれぞれ1〜10で記録

#### 月経周期（女性のみ）
- 周期の開始日・終了日・症状を記録
- コンディションとの相関分析に活用されます

### 2. ACWRを見よう

ホーム画面に表示される **ACWR（急性・慢性ワークロード比）** は、怪我リスクの指標です。

| ACWR値 | 状態 | 色 |
|--------|------|-----|
| 0.8〜1.3 | 理想的 | 緑 |
| 1.3〜1.5 | 注意 | 黄 |
| 1.5以上 | 高リスク | 赤 |
| 0.8未満 | トレーニング不足 | 青 |

ACWRは直近7日間の負荷（急性負荷）÷ 直近28日間の平均負荷（慢性負荷）で計算されます。
急に練習量を増やすとACWRが上がり、怪我のリスクが高まります。

### 3. パフォーマンス測定

メニューの「パフォーマンス」から、各種測定結果を記録できます。

- **ジャンプ**: CMJ、スクワットジャンプ、DJ RSIなど
- **筋力**: ベンチプレス、スクワット、デッドリフト（推定1RMを自動計算）
- **スプリント**: 5m〜50m走
- **持久力**: シャトルラン、Yo-Yoテスト、クーパーテスト
- **アジリティ**: アローヘッド、プロアジリティ、0-5-0

自己ベストを更新すると通知でお祝いされます！

### 4. ゲーミフィケーション

楽しく続けるための仕組みがあります。

- **ストリーク**: 連続記録日数。途切れないように毎日記録しよう！
- **ポイント & レベル**: 記録するとポイント獲得。レベルアップでランクが上がります
- **バッジ**: 特別な条件を達成するとバッジを獲得（7日連続、自己ベスト更新など）
- **ランキング**: チーム内で記録数やポイントを競い合えます

### 5. メッセージ

ヘッダーの 💬 アイコンからコーチやチームメンバーとメッセージのやり取りができます。

### 6. 通知

ヘッダーの 🔔 アイコンでコーチからの一斉通知を確認できます。
Push通知をONにすると、アプリを閉じていても通知を受け取れます。

### 7. オフライン対応

体育館やグラウンドなど電波が弱い場所でも記録できます。
オフラインで入力したデータは、ネットに繋がったら自動的に送信されます。

### 8. 設定

メニュー → 設定 から以下を変更できます。
- プロフィール（名前、身長、生年月日、電話番号）
- プッシュ通知のON/OFF
- ダークモード

## コーチ向けガイド

### 1. チーム立ち上げ

Bekutaでチームを立ち上げる手順はシンプルです。

1. [bekuta.netlify.app](https://bekuta.netlify.app) で「無料で始める」
2. 基本情報とチーム情報を入力
3. 招待リンクをLINEやメールで選手に共有
4. 選手がリンクからアカウント作成して自動参加

詳しくは **[チーム立ち上げガイド]** をご覧ください。

### 2. ダッシュボード概要

ログイン後、チームを選択するとダッシュボードが表示されます。

#### アラートサマリー
- ACWR高リスク、オーバートレーニング、チェックイン未提出などを自動検知
- 優先度順に表示されるので、上から対応していきましょう

#### 選手一覧
- 各選手のACWR、最終チェックイン日、コンディションスコアが一覧で確認できます
- 選手をタップすると詳細画面が開きます

### 3. 選手詳細

選手をタップすると以下が確認できます。
- ACWRチャート（推移グラフ）
- トレーニング負荷の履歴
- 体重・睡眠・モチベーションの推移
- パフォーマンス測定の記録

### 4. 通知管理

「通知管理」タブから、チーム全体または選択した選手にPush通知を送信できます。

- タイトルと本文を入力
- 送信先を「全員」または「選択」から選ぶ
- 送信履歴とプッシュ通知登録状況も確認できます

### 5. メッセージ

選手と1対1でメッセージのやり取りができます。
送信時に選手にPush通知が届きます。

### 6. チーム分析

- **チームトレンド**: チーム全体のACWR平均推移
- **ランキング**: チーム内のパフォーマンス比較
- **レポート生成**: データをPDFでエクスポート

### 7. データエクスポート

トレーニングデータをCSV形式でエクスポートして、詳細な分析や記録保管ができます。

## 招待方法

選手やスタッフの招待はシェアリンクで行います。

1. ダッシュボードの設定画面 → 「招待リンク」
2. 選手用リンクまたはスタッフ用リンクをコピー
3. LINEグループやメールで共有
4. リンクを開いた人が名前・メール・パスワードを入力して自動参加

招待リンクには有効期限があります。期限切れの場合は新しいリンクを発行してください。

## よくある質問

**Q: ACWRが表示されません**
A: ACWRの計算には最低7日分のデータが必要です。毎日チェックインを続けてください。

**Q: 練習が休みの日も記録が必要ですか？**
A: はい！RPE=0、時間=0分で「休養日」として記録してください。休養日の記録もACWR計算に重要です。

**Q: オフラインで記録したデータはどうなりますか？**
A: 画面下部に「未送信 ○件」と表示されます。Wi-Fiやモバイルデータに接続すると自動的に送信されます。

**Q: Push通知が届きません**
A: 設定画面でプッシュ通知がONになっているか確認してください。iPhoneの場合はホーム画面にアプリを追加する必要があります。

**Q: パスワードを忘れました**
A: ログイン画面の「パスワードを忘れた方」からリセットできます。

**Q: 機種変更したらデータは引き継げますか？**
A: はい。データはクラウドに保存されています。新しいスマホでログインし、ホーム画面に追加すれば全て引き継がれます。

**Q: 男女や学年でチームを分けたい**
A: Proプラン以上で複数チームを作成できます。

**Q: 選手が卒業した場合は？**
A: 選手のアカウントを「凍結」することで、データは保持したまま非アクティブにできます。
